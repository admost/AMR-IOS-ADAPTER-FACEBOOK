# AMRAdapterFacebook

Facebook Audience Network adapter for AMRSDK. Allows publishers to mediate Facebook Audience Network banner, native, interstitial and video ads in AMRSDK.

## Versioning

Versions follow `MAJOR.MINOR.OWN_VERSION` where `MAJOR.MINOR` tracks the FBAudienceNetwork SDK version and `OWN_VERSION` is the adapter revision.

## Requirements

- iOS 13.0+
- Xcode 15+

## Installation

### Swift Package Manager

In Xcode: **File → Add Package Dependencies** and enter the repository URL:

```
https://github.com/admost/AMR-IOS-ADAPTER-FACEBOOK.git
```

### CocoaPods

Add the following to your `Podfile`:

```ruby
pod 'AMRAdapterFacebook'
```

Then run:

```sh
pod install
```
